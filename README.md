# Video Course Online Store

This project is a Laravel-based e-commerce application designed for selling educational video courses on physical media.

The platform allows users to browse available products, add items to the cart, and manage their order before completing the purchase process.

## Features

- Product catalog
- Shopping cart
- Product detail pages
- Order management structure
- Simple and clear checkout flow (without integrated online payment)
- Backend structure for managing products and orders

## Technology Stack

- PHP
- Laravel
- MySQL
- Blade templates
- MVC architecture

## Architecture

The project follows Laravel's MVC structure:

- **Controllers** manage application logic and user requests
- **Models** handle database interaction
- **Views** display the product catalog and shopping cart interface

The code is structured to keep controllers readable and separate business logic where possible.

## Purpose of the Project

This project demonstrates:

- building an e-commerce backend with Laravel
- implementation of a shopping cart
- product catalog structure
- clean and maintainable code organization
